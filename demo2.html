browse the map and share your location:


  • browse the map...

  • select Share Location from the main map menu

  • select Ok to display the pin on the map

  • move the pin to your festival location, for example the Greenpeace area

  • enter how long you will be at this location and write a message

  • select the friends you want to share your location with

  • and select Send from the Options menu
